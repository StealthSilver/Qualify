# Qualify - Landing Page

A modern, premium animated landing page for Qualify, an ed-tech platform for JEE/NEET aspirants.

## 🚀 Features

- **Next.js 16** with App Router
- **TypeScript** for type safety
- **TailwindCSS 4** for styling
- **Framer Motion** for smooth animations
- **Lucide React** for beautiful icons
- Fully responsive design
- Dark theme optimized
- Performance optimized

## 🎨 Design Philosophy

The landing page embodies a next-gen immersive ed-tech SaaS product with:

- Dark, sophisticated color palette (#070a05 background)
- Animated textured backgrounds with subtle particles
- Scroll-based animations and transitions
- Glassmorphism effects
- Gradient accents (indigo, purple, pink)
- Premium micro-interactions

## 📦 Installation

```bash
npm install
```

## 🏃 Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the landing page.

## 🏗️ Build

```bash
npm run build
npm start
```
